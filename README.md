E-Commerce Automation Testing Framework
Project Overview
This project is a Selenium Java Test Automation Framework developed for an E-Commerce web application. It automates end-to-end user workflows such as login, product search, cart management, and checkout.

Tech Stack
Java
Selenium WebDriver
TestNG
Maven
Page Object Model (POM)
Features Automated
User Login & Signup
Product Search
Add to Cart
Checkout Flow
Form Validation
Framework Design
Page Object Model (POM)
Reusable components
Explicit waits for stability
Screenshot capture on failure
How to Run
Clone the repository
Open in Eclipse / IntelliJ
Run testng.xml
